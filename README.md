# Rabbit Followers

A simple React + TypeScript app where rabbits follow your mouse cursor around the screen.

## Prerequisites

- [Bun](https://bun.sh/) installed on your system

## Installation

```bash
cd rabbit-followers
bun install
```

## Running the App

```bash
bun dev
```

The app will start at [http://localhost:5173/](http://localhost:5173/)

## How to Use

Simply move your mouse around the screen and watch the rabbits chase after it! Each rabbit moves at a different speed, creating a fun trailing effect.
